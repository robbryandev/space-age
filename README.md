# [Space Age Calculator](https://robbryandev.github.io/space-age/)

## Contributers
* Robert Bryan

## Description
A web app to calculate your age given a different amount of days in a year.

## Technologies Used
* HTML
* CSS
* Bootstrap
* Animate.css
* Javascript
* Typescript
* Webpack
* JQuery

## Setup/Installation Requirements
### Requirements
* Nodejs

### Setup
* Download repo
* Install packages
```js
npm i
```
* Build site
```js
npm run build
```
* Open dist/index.html in your browser
